北京大学分子影像实验室

北京大学分子影像实验室立足于光学与核医学分子影像领域，研究内容贯穿分子影像设备研发、活体成像和转化应用、影像数据分析处理的全过程，聚焦于多模态成像技术、眼科光学成像和小动物荧光成像等具体问题。
